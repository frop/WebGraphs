Data - Diretório que armazena os dados
	Graphs - Diretório que armazena os arquivos de grafos
	Results - Diretório que armazena os arquivo de resultados
Library - Biblioteca do sistema
	Algorithms - Diretório onde ficam os arquivos de algoritmos
		AlgorithmBase.php - Classe base para todos os algoritmos
	Resources - Diretório onde ficam os arquivos responsáveis por implementar cada recurso
		Algorithm.php - Classe para tratar o recurso ALGORTIHM
		Graph.php - Classe para tratar o recurso GRAPH
		Result.php - Classe para tratar o recurso RESULT
	RestService.php - Classe responsável por controlar e responder todas as requisições
	GraphBase.php - Classe que manipula um grafo
Public - Diretório acessível pelo servidor web
	index.php - Arquivo que recebe todas as requisições
	.htaccess - Arquivo responsável por redirecionar as requisições
HOWTO - Arquivo que explica como colocar o sistema no ar
ClientExamples - Diretório que armazena alguns arquivos com exemplos de clientes
